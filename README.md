### automation-with-playwright

This is a simple web automation project using playwright to run the tests. In this project, development techniques in BDD, Factory Pattern and Page Object Model (POM) are applied.

### Technologies:

- Java 17
- Maven
- Playwright
- Cucumber
- Lombok
- JUnit
- Docker
